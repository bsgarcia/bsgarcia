RL Space Shooter

×
The bandit task is an experimental translation of the multi-armed bandit problem. Here subjects are presented with different pairs of options, and must find which option within each pair maximizes their gains. We (with Sophie Bavard and Stefano Palminteri) aimed at gamifying this class of tasks, by designing it as a space shooter using unity game engine. Two assumptions motivated us:

(i) a task with a gameplay closer to actual video games might increase subject engagement and reduce behavioral noise.
(ii) a corollary is that it becomes a video game, which in a sense is more ecological as playing video games is a common practice that involves 'real-life virtual decisions'.

Multiplayer Public Good Game

×
A public good game is a standard design in behavioral economics where subjects are asked to contribute to a public pot. Their contribution is thereafter multiplied by a factor, and the total is distributed among the players. In this particular version, we (with Sacha B.-Gironde, Fernanda Sanko, Theresa Pachmann) made an online multiplayer public good game (using Otree), where subjects are asked whether they want to display (to the other player) their factor prior to contribute. We hypothesized that reinforcing trust through this disclosure mechanism would allow greater contributions. The aesthetic of the game is (very) inspired by Nicky Case productions.
